Introduction

Welcome to the KNU ERP Management System User Manual. This documentation is designed to guide students, faculty, and administrators through the various features of the platform.

Tip: Use the sidebar on the left to navigate quickly between different modules. Click on any image to zoom in for a better view.

Dashboard Overview

The dashboard acts as the central hub for all ERP activities. It provides a quick summary of recent activities and notifications.

Student Onboarding

This section handles the initial data synchronization. Student data is pulled from the central government portal and populated into the KNU ERP system automatically.

Important: Ensure internet connectivity is stable during the onboarding synchronization process.

Student Portal Login

Students can access their personal dashboard using their unique credentials.

Login Methods

  • Option 1:
    User: Student ID or App Number
    Pass: Application Code
  • Option 2 (No ID):
    User: Registered Phone Number
    Pass: Application Code

Completing Registration

Once logged in, students must complete the academic registration form.

Step 1: Dashboard Navigation

Click on the "Registration Form" button visible on the main card.

Step 2: Academic Year Selection

Select the current academic session carefully.

Step 3: Form Details

Fill in all required fields as shown in the sample below.

Fee Payment

After form submission, the payment gateway becomes active.

Select your preferred payment method (UPI, Card, Netbanking) and complete the transaction.

After payment, a receipt will be generated. Please Download and keep it safe.

Principal / HOD Module

Administrative heads use this module to verify and approve student applications.

Approval Workflow

  1. Navigate to Registration > Approve Students.
  2. Use the search filters to find specific batches.
  3. Select students using checkboxes.
  4. Choose an action: Approve, Return, or View.
